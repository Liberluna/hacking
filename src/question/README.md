# Question
## 追加方法
このディレクトリにある、`question.json`を編集してください。
`"questions"`配列の中に、オブジェクトを追加していきます。
オブジェクトの書き方は、以下の通りです。
- `"title"`(必須): string ... 重複できる問題のタイトルです。
- `"id"`(必須): string ... 重複しない問題のIDです。
- `"level"`(必須): string ... 難易度です。重複できます。
- `"url"`(必須): string ... 問題へのURLです。
- `"releases"`(必須): id[] ... このステージをクリアすることによって解放されるステージのIDたちです。
